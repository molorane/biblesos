390 Le Chabile le Chabile - F. Coillard

1 Le chabile, le chabile,
Letsatsi le letle!
Ke letsatsi la phomolo,
La kgotso, la thabo.
O, le monate hakaakang,
Tsatsi lena le ratehang!
Thabang bohle: le chabile,
Letsatsi le letle!
Le chabile, le chabile,
Letsatsi le letle!
Ke letsatsi la phomolo,
La kgotso, la thabo.

2 Le chabile, le chabile,
Letsatsi le letle!
Le hlakole mofufutso,
Le dule ka kgotso.
Le lebale tsa lefatshe,
Pelaelo ditsietsi.
Le ikhutseng, le thabele
Letsatsi le letle!
Le chabile, le chabile ...

3 Le chabile, le chabile,
Letsatsi le letle!
Itokiseng ka monyaka
Ho ya ho Morena;
Ho mo fapha ka dithoko,
Ntat'a lona ea lerato.
Le se tsoafe, le chabile,
Letsatsi le letle!
Le chabile, le chabile ...

4 Le chabile, le chabile,
Letsatsi le letle!
E, re tla dula kajeno
Maotong a Jesu,
Ntate, buisa ngoan'a hao,
Ka tsa mosa, tsa mohau.
O, Mong'a ka, se ntebale
Ka tsatsi le letle!
Le chabile, le chabile ...

5 Le leng le teng, le kapele,
Letsatsi le letle,
La phomolo e sa feleng,
La kgotso e tletseng.
Ho ea kang 'na, ea kgathalang.
Na nyakallo e tla ba joang,
Ha ke bona ho chabile
Letsatsi le letle!
Le chabile, le chabile ...
