102 Le Monate, le Monate - A. Mabille

1 Le monate, le monate,
Letsatsi lane ke dumelang
Ho 'Moloki ea ntopolotseng,
Ea hlakolang dibe tsohle.
Le letle, le letle,
Tsatsi lane la nyakallo.
Mor'a Ntate, kea o rapela,
Moren'a ka, kea o leboha.
Le letle, le letle,
Tsatsi lane la nyakallo.

2 Le monate, le monate,
La kajeno le sa nchabelang;
Ke disitswe, ke ea bolokoang
Lefatsheng lena ke Ntate.
Le letle, le letle,
Tsatsi lena la nyakallo.
Mor'a Ntate, kea o rapela;
Moren'a ka, kea o leboha.
Le letle, le letle,
Tsatsi lena la nyakallo.

3 Le monate, le monate,
Tsatsi le tlang, le sa tl'o fela,
La dithoko le la difela
Tsa ho roka Mor'a Ntate.
Le letle, le letle,
Tsatsi lela la nyakallo.
Mor'a Ntate, kea o rapela;
Moren'a ka, kea o leboha.
Le letle, le letle.
Tsatsi lela la nyakallo.
